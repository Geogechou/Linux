### 整数的输出

|          | short | int  | long |
| -------- | :---- | ---- | ---- |
| unsigned | %hu   | %u   | %lu  |
| signed   | %hd   | %d   | %ld  |

### 整数相关的的库函数

* atoi(const char* str)函数

  将字符串转化为int

* atol(const char* str)

  将字符串转化为long

* labs(long num)

  将long类型取绝对值

* srand(long num)、rand()取随机数
